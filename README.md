# Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics workflow using Python, SQL databases, and Power BI.  
The main objective of the project is to analyze raw data, clean and transform it, perform exploratory data analysis (EDA), execute SQL queries across multiple database systems, and build interactive dashboards for business insights.

The project also includes a final analytical report and presentation created using Gamma.

---

## Dataset
The dataset used in this project contains structured business data for analysis purposes.

### Dataset Features
- Multiple numerical and categorical columns
- Missing values and duplicates
- Business-related KPIs
- Time-based and category-based data

### Data Source
- CSV / Excel dataset
- Imported into Python and SQL databases

## Project Workflow

### 1. Data Loading
- Imported dataset using Python
- Verified structure and data types
- Handled initial formatting

### 2. Exploratory Data Analysis (EDA)
- Checked missing values
- Identified duplicates
- Analyzed distributions and trends
- Created visualizations for insights

### 3. Data Cleaning
- Removed duplicates
- Filled or handled null values
- Standardized column formats
- Corrected inconsistent data

### 4. SQL Analysis
Performed SQL queries using:
- PostgreSQL
- MySQL
- SQL Server

#### SQL Tasks Included
- Filtering and sorting
- Aggregations
- Joins
- Group By operations
- Window functions
- KPI calculations

### 5. Dashboard Development
Created an interactive Power BI dashboard including:
- KPI cards
- Trend analysis
- Category-wise insights
- Filters and slicers
- Interactive charts

### 6. Reporting
- Prepared a detailed analytical report
- Summarized findings and business insights
- Added charts and conclusions

### 7. Presentation
- Created a professional PPT using Gamma
- Included project overview, methodology, dashboard screenshots, and insights
